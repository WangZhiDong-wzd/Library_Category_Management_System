# 在线书城后台管理系统

> 一个基于 **jQuery + Bootstrap + Chart.js** 构建的轻量级纯前端图书管理后台 SPA。

---

## 📋 项目概览

| 字段 | 内容 |
|------|------|
| **项目名称** | 在线书城后台管理系统 |
| **项目类型** | 纯前端单页应用（SPA） |
| **技术栈** | jQuery 3.7.1 · Bootstrap 4.6.2 · Chart.js 4.4.0 · Font Awesome 6 |
| **数据存储** | LocalStorage + SessionStorage（无后端依赖） |

| **开发工具** | VS Code |

## 🎯 核心功能模块

### 1. 🔐 登录系统

- 双模式登录：**用户名登录** 和 **手机号登录**
- 表单实时校验（非空检查、手机号格式、密码最小长度）
- 密码可见性切换（眼睛图标）
- 加载状态动画模拟

### 2. 📊 数据仪表盘

- **四大统计卡片**：图书总数 · 订单总数 · 注册用户 · 总营收
- **柱状图**：各图书类别销售数量对比（Chart.js）
- **折线图**：近 7 日订单状态趋势（Chart.js）
- **最新订单列表**：最近 5 条订单快捷查看

### 3. 📚 图书管理

- 图书信息 **CRUD**（增删改查）
- 按图书名称 / ISBN 搜索
- 图书类别 **CRUD**（添加、编辑、搜索、删除）
- 默认 4 大类别：文字 · 化学 · 历史 · 计算机

### 4. 👤 用户管理

- 用户信息 **CRUD**（增删改查）
- 按用户名 / 邮箱搜索
- 用户等级 **CRUD**（普通会员 · 高级会员 · VIP 会员）
- 支持编辑用户资料

### 5. 📦 订单管理

- **订单总览**：全部订单统计卡片 + 状态筛选 + 搜索
- **待处理订单**：专属统计（待处理数 · 今日新增 · 加急订单）
- 状态流转：待处理 → 已发货 → 已完成 / 已取消
- 支持按时间 / 总价排序
- 订单详情弹窗查看

## 💡 关键技术亮点

### 数据持久化策略

| 数据类型 | 存储方式 | 说明 |
|---------|---------|------|
| 图书类别 | `sessionStorage` | 会话级，关闭标签页自动清除 |
| 用户等级 | `sessionStorage` | 会话级，关闭标签页自动清除 |
| 订单数据 | `localStorage` | 持久化，跨会话保留 |
| 用户数据 | `localStorage` | 持久化，跨会话保留 |

> 采用分层存储策略：配置类数据使用 `sessionStorage`（避免错误配置持久化），核心业务数据使用 `localStorage`（保证数据不丢失）。

### 交互设计

- **手风琴式菜单**：同一时间只展开一个二级菜单，界面清晰
- **悬停高亮 + 点击高亮**：多级菜单状态管理
- **统计卡片 hover 上浮动画**：提升交互质感
- **Chart.js 入场动画**：柱状图弹性反弹 + 折线图平滑过渡
- **表单实时校验**：输入即校验，错误即时清除
- **模态弹窗**：Bootstrap 模态框实现图书/类别/等级 CRUD
- **订单状态徽章**：彩色标签区分订单状态

### 图表可视化（Chart.js）

- **柱状图**：4 类图书销售对比，独立配色，圆角柱顶
- **折线图**：4 种订单状态 7 日趋势，差异化线型（实线 + 虚线）+ 差异化数据点形状（圆形 · 方形 · 三角形）

### 无障碍访问（ARIA）

- `role="menubar"` / `role="menuitem"` 导航语义标注
- `aria-haspopup` / `aria-expanded` 子菜单状态提示
- `role="tablist"` / `role="tab"` 标签页无障碍支持
- `role="alert"` 实时错误播报
- `aria-label` / `aria-labelledby` / `aria-describedby` 全面标注

## 🚀 本地运行

```bash
# 使用 VS Code Live Server（推荐）
code .
# 右键 index.html → Open with Live Server

# 或使用 Python 简易服务器
python -m http.server 8080

# 或使用 Node.js
npx serve .
```

打开浏览器访问 `http://localhost:8080/login.html` 即可进入登录页。

> **登录验证**：用户名 + 密码任意非空即可（前端模拟登录），跳转到后台首页。

## 📝 总结

本项目是一个 **纯前端、零后端依赖** 的图书管理后台系统，采用 **jQuery + Bootstrap** 经典组合，配合 **Chart.js** 实现数据可视化。所有数据通过 Web Storage API 在浏览器端完成持久化，开箱即用，无需任何服务器配置。

项目代码组织清晰，每个功能模块独立成页，注释完善（含数据流说明、架构决策记录），CSS 样式模块化隔离，充分体现了前端工程化的基础素养。
